COMICVERSE STORE - SITIO WEB ACADÉMICO

Tema: Tienda de cómics
Grupo: 14
Tecnologías: HTML5 + CSS3
Base de datos: No utiliza
Frameworks: No utiliza Angular, React ni Vue

ARCHIVOS PRINCIPALES:
- index.html
- productos.html
- promociones.html
- sucursales.html
- styles.css
- img/

INSTRUCCIONES PARA ABRIR:
1. Extraer/descomprimir completamente el archivo ZIP.
2. Abrir index.html en Google Chrome o Microsoft Edge.
3. Navegar por Productos, Promociones y Sucursales.

INSTRUCCIONES PARA SUBIR A GITHUB PAGES:
1. Crear un repositorio en GitHub, por ejemplo: comicverse-store.
2. Subir todos los archivos y carpetas del proyecto, no el ZIP.
3. Entrar a Settings > Pages.
4. En Source seleccionar Deploy from a branch.
5. Elegir branch main y carpeta /root.
6. Guardar y esperar el enlace público.

OBSERVACIÓN:
El diseño fue preparado para cumplir una práctica universitaria de Desarrollo de Aplicaciones Web con cuatro páginas: principal, productos, promociones y sucursales.
