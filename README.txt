ComicVerse Store

Proyecto web listo para GitHub Pages.

Estructura:
- index.html
- productos.html
- promociones.html
- sucursales.html
- styles.css
- carpeta img

Instrucciones:
1. Subir todos los archivos y la carpeta img al repositorio.
2. Verificar que index.html quede en la raíz del repositorio.
3. Activar GitHub Pages desde Settings > Pages.
4. Esperar unos minutos y actualizar el navegador con Ctrl + F5.

Nota técnica:
Las imágenes están separadas en la carpeta img para que los archivos HTML sean livianos y GitHub pueda mostrar el código normalmente.
